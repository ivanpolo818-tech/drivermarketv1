## 🚀 Cómo ejecutar

### Paso 1: Instalar
```bash
python setup_docs.py install
```

### Paso 2: Ver en local
```bash
python setup_docs.py serve
# Abre http://localhost:8000
```

### Paso 3: Compilar
```bash
python setup_docs.py build
```

### Paso 4: Desplegar (opcional)
```bash
python setup_docs.py deploy
# GitHub Pages
```
