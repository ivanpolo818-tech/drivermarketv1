    4. Flujo de Autenticación OAuth Google¶

    sequenceDiagram
    participant Browser as Navegador Usuario
    participant Flask as Servidor Flask
    participant Google as Google OAuth
    participant DB as PostgreSQL

    Browser->>Flask: Clic en "Login con Google"
    Flask->>Google: Redirige a Google OAuth
    Google->>Browser: Formulario de login Google
    Browser->>Google: Credenciales usuario
    Google->>Browser: Código de autorización
    Browser->>Flask: Código + State
    Flask->>Google: Intercambia código por token
    Google->>Flask: ID Token + email
    Flask->>DB: ¿Usuario existe?
    alt Usuario Existe
        DB-->>Flask: Datos usuario
    else Usuario Nuevo
        Flask->>DB: INSERT nuevo usuario
        DB-->>Flask: Usuario creado
    end
    Flask->>Browser: Cookie Session
    Browser->>Flask: Acceso a /users/dashboard
    Flask->>Browser: Dashboard cargado

    7. Diagrama de Transacciones de Base de Datos¶

    sequenceDiagram
    participant App as Aplicación Flask
    participant Conn as Conexión PostgreSQL
    participant DB as PostgreSQL (Transacción)

    App->>Conn: Begin Transaction
    Conn->>DB: BEGIN

    App->>DB: INSERT INTO usuarios
    Note right of DB: Fila insertada (no visible aún)

    App->>DB: INSERT INTO perfil_vendedor<br/>(usuario_id = new_id)
    Note right of DB: Ambas inserciones<br/>no confirmadas

    App->>Conn: ¿Validación OK?
    alt Validación Exitosa
        Conn->>DB: COMMIT
        DB-->>App: ✅ Cambios persistidos
    else Error de Validación
        Conn->>DB: ROLLBACK
        DB-->>App: ❌ Todo revierte
    end
