    📖 Documentación Técnica Drivemarket¶

    Documentación completa para desarrolladores de la plataforma Drivemarket

    🚀 Inicio rápido (3 pasos)¶

    1️⃣ Instalar dependencias¶

    python setup_docs.py install

    2️⃣ Ver en navegador¶

    python setup_docs.py serve
    Abre: http://localhost:8000

    3️⃣ Compilar para producción¶

    python setup_docs.py build

    🔧 Tecnologías documentadas¶

    Backend:
  - Flask 2.3.3 (web framework)
  - SQLAlchemy 3.0.5 (ORM)
  - PostgreSQL 12+ (database)
  - Authlib 1.6.9 (OAuth)
  - Flask-Mail 0.10.0 (email)
  - OpenAI API (chatbot)

Documentación:
  - MkDocs 1.5.3 (site generator)
  - Material Theme (professional UI)
  - Mermaid.js (diagramas)
  - Markdown (content)

    🚀 Desplegar en producción¶

    GitHub Pages (recomendado)¶

    python setup_docs.py deploy
    Sitio en: https://username.github.io/drivemarket/

    Otro servidor¶

    # 1. Compilar
python setup_docs.py build

# 2. Copiar contenido de site/ a tu servidor
scp -r site/* usuario@servidor:/var/www/docs/

    Netlify¶

    Conectar repo en Netlify con: - Build command: mkdocs build - Publish directory: site

    📞 Problemas frecuentes¶

    Python no encontrado¶

    # Windows
py setup_docs.py install

# Mac/Linux
python3 setup_docs.py install

    Puerto 8000 en uso¶

    mkdocs serve --dev-addr 127.0.0.1:8001

    PDF no se genera¶

    Requiere Chrome instalado. Descárgalo en https://google.com/chrome/

    Cambios no aparecen¶

    Recarga el navegador (Ctrl+Shift+R) o reinicia el servidor
